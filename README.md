# XCode-Cheatsheet

## Zoom
- Alt + Mouse Scroll

## Open and Close Navigation Pane
- CMD + 0

## Open and Close Utility Pane
- CMD + Option + 0

## Open and Close Debug Pane
- CMD + y

## Run Application
- CMD + r


## Open Assistant Editor
- CMD + Enter

## Close Assistant Editor
- CMD + Option + Enter

## Show Info (Documentation) for item
- Option click 

## Rename IBOutlet
- CMD + click
